# 构图裁剪数据集构建项目说明

## 1. 当前已实现内容

本项目已经新增一个可执行数据集构建包：

```text
composition_dataset_builder/
```

它可以完成：

```text
原图读取
caption / semantic router
VLM 结构化理解，支持 heuristic / precomputed / OpenAI Responses / Qwen
YOLO 或 VLM bbox 检测主体和关键物
SAM 或 bbox mask 分割关键区域
crop-state graph 构建
GAIC-style grid candidates
方向性 candidates
现有 caption-rule-co 规则 candidates
负样本 candidates
多维候选评分排序
JSONL 数据保存
top-k crop 保存
mask 保存
可视化图保存
summary 报告保存
```

## 2. 最小运行命令

不依赖 OpenAI、Qwen、YOLO、SAM、TorchEAT 的 MVP：

```bash
python -m composition_dataset_builder.cli \
  --image-root caption-rule-co/test \
  --captions caption-rule-co/gemini_captions.json \
  --out-dir runs/composition_dataset_mvp \
  --target-aspects original,1:1,4:5 \
  --max-images 20
```

只测试新候选逻辑，不接入旧规则：

```bash
python -m composition_dataset_builder.cli \
  --image-root caption-rule-co/test \
  --captions caption-rule-co/gemini_captions.json \
  --out-dir runs/composition_dataset_mvp_no_old_rules \
  --target-aspects original,4:5 \
  --max-images 20 \
  --no-existing-rules
```

## 3. 推荐运行命令

### 3.1 使用 OpenAI `/v1/responses` 做 VLM teacher

```bash
export OPENAI_API_KEY="你的 key"

python -m composition_dataset_builder.cli \
  --image-root caption-rule-co/test \
  --captions caption-rule-co/gemini_captions.json \
  --out-dir runs/composition_dataset_openai \
  --target-aspects original,4:5 \
  --vlm openai \
  --openai-model gpt-4.1-mini \
  --openai-image-detail auto \
  --detector vlm \
  --segmenter bbox
```

也可以直接运行：

```bash
bash scripts/build_dacc_with_openai.sh
```

OpenAI 输出用于：

```text
主体识别
关键物体识别
关系理解
重要背景判断
干扰物判断
构图问题判断
建议裁剪动作
可选 bbox_norm
```

### 3.2 使用 Qwen 做 VLM teacher

```bash
export DASHSCOPE_API_KEY="你的 key"

python -m composition_dataset_builder.cli \
  --image-root caption-rule-co/test \
  --captions caption-rule-co/gemini_captions.json \
  --out-dir runs/composition_dataset_qwen \
  --target-aspects original,4:5 \
  --vlm qwen \
  --qwen-model qwen-vl-plus
```

Qwen 输出用于：

```text
主体识别
关键物体识别
关系理解
重要背景判断
干扰物判断
构图问题判断
建议裁剪动作
可选 bbox_norm
```

### 3.3 使用 YOLO 做检测

```bash
python -m composition_dataset_builder.cli \
  --image-root caption-rule-co/test \
  --captions caption-rule-co/gemini_captions.json \
  --out-dir runs/composition_dataset_yolo \
  --target-aspects original,4:5 \
  --detector yolo \
  --yolo-model caption-rule-co/yolo11n.pt
```

### 3.4 使用 SAM 做精细 mask

```bash
python -m composition_dataset_builder.cli \
  --image-root caption-rule-co/test \
  --captions caption-rule-co/gemini_captions.json \
  --out-dir runs/composition_dataset_sam \
  --target-aspects original,4:5 \
  --detector yolo \
  --yolo-model caption-rule-co/yolo11n.pt \
  --segmenter sam \
  --sam-checkpoint /path/to/sam_vit_h.pth
```

如果没有安装 `segment_anything` 或没有 checkpoint，代码会回退到 bbox mask，不会中断主流程。

### 3.5 使用 TorchEAT 做美学评分

```bash
python -m composition_dataset_builder.cli \
  --image-root caption-rule-co/test \
  --captions caption-rule-co/gemini_captions.json \
  --out-dir runs/composition_dataset_torcheat \
  --target-aspects original,4:5 \
  --aesthetic torcheat \
  --aesthetic-model caption-rule-co/TorchEAT.pth
```

## 7. 已修复的工程问题

已为本地旧项目目录补充：

```text
caption-rule-co/rules/__init__.py
caption-rule-co/semantic/__init__.py
```

原因：环境中存在第三方 `rules` 包，如果本地 `rules` 没有 `__init__.py`，Python 可能错误导入第三方包，导致旧规则调度器不可用。

## 9. 下一步建议

建议按这个顺序继续增强：

```text
1. 用默认模式跑 100-500 张图，抽查 JSONL 和可视化。
2. 接入 YOLO，确认 subject mask 和 relation mask 质量。
3. 接入 OpenAI Responses 或 Qwen，确认 vlm_understanding 的 semantic_type、action、issue 是否合理。
4. 接入 SAM，把 bbox mask 替换成精细 mask。
5. 接入 TorchEAT 或 VLM pairwise 评分，调 candidate scoring 权重。
6. 抽样人工复核 top-1/top-5，形成人工偏好集。
7. 用 all.jsonl 训练第一版 crop ranker。
```
