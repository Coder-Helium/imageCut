# 数据结构说明

本文档描述 `composition_dataset_builder` 输出 JSONL 的核心结构。

## 1. 样本粒度

一个样本对应：

```text
一张图片 + 一个目标裁剪比例
```

同一张图片如果生成 `original,1:1,4:5` 三个比例，就会产生三行 JSONL。

## 2. 顶层字段

```json
{
  "sample_id": "000001__4x5",
  "image_path": "/abs/path/000001.jpg",
  "rel_path": "subdir/000001.jpg",
  "image_width": 1920,
  "image_height": 1280,
  "target_aspect_ratio": "4:5",
  "caption": "...",
  "semantic_type": "person_holding_object",
  "semantic_info": {},
  "vlm_understanding": {},
  "detections": [],
  "masks": {},
  "crop_state_graph": {},

  "best_score": 4.62,
  "best_action": "preserve_relation",
  "main_issue": "subject_object_relation_should_be_preserved",
  "quality_flags": {}
}
```

```json
  "main_subject": {

    "category": "person",

    "importance": 1.0,
    "bbox_norm": [0.16, 0.09, 0.51, 0.96]
  },
  "key_objects": [
    {
      "name": "camera",

      "relation_to_subject": "held_by_subject",
      "importance": 0.85,
      "bbox_norm": [0.34, 0.34, 0.43, 0.46]

  "important_background": [],

```

## 4. detections

检测器输出统一格式：

```json
{
  "name": "person",
  "category": "person",
  "bbox": [310, 120, 980, 1240],
  "confidence": 0.92,
  "source": "yolo"
}
```

如果没有 YOLO，也可以从 Qwen 的 `bbox_norm` 生成 detection。

## 5. masks

mask 分成四类：

```text
preserve_masks      主体或必须保留区域
relation_masks      关键物体/关系物体
environment_masks   重要环境区域
distractor_masks    可裁掉干扰区域
```

示例：

```json
{
  "mask_id": "000001_00_person",
  "name": "person",
  "category": "person",
  "role": "preserve",
  "bbox": [310, 120, 980, 1240],
  "mask_path": "masks/000001/000001_00_person.png",
  "area": 750400,
  "area_ratio": 0.305,
  "confidence": 0.92,
  "source": "bbox_mask",
  "importance": 1.0
}
```

## 8. 训练用法

### Crop Ranker

使用：

```text
image + candidates[*].box -> candidates[*].scores.final_score
```

### End-to-end Crop Generator

使用：

```text
image -> top-k candidates 或 best_crop
```

### Direction-aware Model

使用：

```text
image -> main_issue + best_action + best_crop
```
